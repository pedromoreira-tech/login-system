# Sistema de Login em PHP

## 📌 Descrição
Sistema de autenticação com cadastro, login e sessão usando PHP e MySQL.

## 🚀 Funcionalidades
- Cadastro de usuário
- Login
- Logout
- Página protegida (dashboard)

## 🛠 Tecnologias
- PHP
- MySQL
- HTML/CSS

## ▶️ Como rodar
1. Clonar o repositório
2. Importar o arquivo SQL
3. Rodar em XAMPP/Laragon
4. Acessar: http://localhost/login-system/public/
